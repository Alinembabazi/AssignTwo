Kigali City Guide

Project Description
A comprehensive travel website that showcases things to do and places to eat around Kigali, Rwanda. This website serves as an informative guide for both visitors and locals, featuring key attractions, cultural sites, and scenic destinations in Rwanda's capital city.

HTML Pages Included
1. index.html - Home page with navigation and overview of Kigali attractions
2. kcc.html - Kigali Convention Centre destination page
3. genocide-memorial.html - Kigali Genocide Memorial destination page
4. mount-kigali.html - Mount Kigali destination page
5. inema-art.html - Inema Art Centre destination page

About Kigali

Kigali is a modern and vibrant city known for its rich history, cultural creativity, and beautiful natural landscapes. This website highlights four key attractions:

-Kigali Convention Centre: Represents the city's development and international presence
-Kigali Genocide Memorial: An important site for remembrance and education
-Inema Art Centre: A hub for contemporary Rwandan art and cultural expression
-Mount Kigali: A popular destination for outdoor activities and scenic views
